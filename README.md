# Financial Ratio Analysis Project (IAG-AU vs Competitor)

Dự án Python phân tích và so sánh các chỉ số tài chính (ROE, DE) giữa **Insurance Australia Group (IAG)** và đối thủ cạnh tranh dựa trên dữ liệu từ FactSet.

## Chức năng chính
* Đọc và xử lý dữ liệu từ file Excel (`.xlsx`).
* Xử lý dữ liệu tài chính (ROE, Debt/Equity) qua 15 năm.
* Vẽ biểu đồ đường so sánh hiệu quả sử dụng vốn (ROE).
* Tính toán các chỉ số thống kê (Mean, Standard Deviation).

## Công nghệ sử dụng
* Python 3.x
* Thư viện: Pandas, Matplotlib, Openpyxl.

## Cách chạy
1. Cài đặt thư viện: `pip install pandas matplotlib openpyxl`
2. Chạy file chính: `python main.py`
